 
## Server Api

[Nest](https://docs.nestjs.com/) framework and [PostgreSql](https://node-postgres.com/) plugin

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## Test

```bash
$ npm run test
```
